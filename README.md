# holynoter

A Bible app for reading a taking notes.

[Demo](https://jkaldenbach.github.io/holynoter/)

# Install / Run

```
npm install
npm start
```

# Features

* Passage lookup
* Search history
* Click on a verse to
  * Make notes
  * Highlight
  * Circle
  * Underline
* All data is stored locally
